# Harvester Icon

Please place your harvester/tractor icon image here as `harvester.png`

## Requirements:
- Filename: `harvester.png`
- Format: PNG with transparent background
- Size: 42x42px or larger (will be scaled to 42px)
- Color: Flat green design
- Style: Simple, modern icon representing a tractor/harvester

## File Path:
`public/assets/icons/harvester.png`

The loader component will display this icon in the center of the spinning ring animation.
